# Ingeniería de videojuegos con Godot: recursos

## Knowledge

- [Conceptos clave de Godot](https://docs.godotengine.org/en/stable/getting_started/introduction/key_concepts_overview.html)
  Fuente oficial sobre nodos, escenas, árbol de escenas y señales. Usar para construir el modelo mental del motor.
- [Nodos y escenas](https://docs.godotengine.org/en/stable/getting_started/step_by_step/nodes_and_scenes.html)
  Explicación oficial de las unidades con las que se compone un juego y de cómo Godot elige la escena principal.
- [Curso oficial paso a paso](https://docs.godotengine.org/en/stable/getting_started/step_by_step/)
  Secuencia introductoria mantenida por Godot. Usar para contrastar nuestras prácticas sobre el proyecto real.
- [Primer script en GDScript](https://docs.godotengine.org/en/stable/getting_started/step_by_step/scripting_first_script.html)
  Introducción oficial a scripts, funciones virtuales y procesamiento por cuadro.
- [Referencia de GDScript](https://docs.godotengine.org/en/stable/tutorials/scripting/gdscript/gdscript_basics.html)
  Fuente técnica para consultar sintaxis y semántica cuando una lección llegue al nivel del código.
- [Buenas prácticas de Godot](https://docs.godotengine.org/en/stable/tutorials/best_practices/index.html)
  Guías oficiales sobre organización de escenas, scripts, datos y proyectos. Usar al refactorizar el prototipo.

## Wisdom (Communities)

- [Foro oficial de Godot](https://forum.godotengine.org/)
  Comunidad moderada para contrastar decisiones, pedir revisión de problemas concretos y aprender de proyectos reales.
- [Comunidades reconocidas por Godot](https://godotengine.org/community/)
  Directorio oficial de espacios de discusión, grupos y eventos. Usar cuando una decisión necesite experiencia práctica diversa.

## Gaps

- Elegir más adelante una fuente primaria sobre matemáticas para juegos 2D cuando el proyecto necesite vectores y movimiento en profundidad.
- Incorporar recursos especializados de diseño de juegos, audio, arte, pruebas y publicación cuando esos subsistemas entren en la ruta.
