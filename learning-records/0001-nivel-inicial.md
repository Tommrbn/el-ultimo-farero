# Nivel inicial y enfoque top-down

El estudiante se identifica como principiante y quiere aprender ingeniería de videojuegos mediante *El Último Farero*. Las próximas sesiones no deben presuponer conocimientos de programación y deben comenzar por el comportamiento y la arquitectura global antes de descender al código.

**Evidence:** declaración directa del estudiante al definir la misión.

**Implications:** el primer umbral de aprendizaje es poder narrar el flujo de ejecución del proyecto con vocabulario sencillo.
