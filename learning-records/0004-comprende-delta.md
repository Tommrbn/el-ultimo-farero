# Comprende delta como tiempo entre frames

El estudiante explicó correctamente que `delta` representa el tiempo transcurrido desde el último frame. También calculó que, con una velocidad de 220 píxeles por segundo y `delta = 0.02`, el desplazamiento del frame es 4.4 píxeles, aunque necesitó separar explícitamente tiempo de distancia.

**Evidence:** definición con sus propias palabras y aplicación numérica correcta.

**Implications:** las próximas explicaciones de movimiento pueden usar `velocidad × delta`, manteniendo visibles las unidades para no confundir segundos con píxeles.
