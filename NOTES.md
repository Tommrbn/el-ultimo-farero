# Teaching notes

- El estudiante se identifica como principiante.
- Quiere comprender el trabajo integral de ingeniería de videojuegos: arquitectura y código incluidos.
- Prefiere aprender top-down: comenzar con el mapa del sistema y descender gradualmente hasta la sintaxis.
- El objetivo práctico y fuente de ejemplos es *El Último Farero*, según `README.md`.
- Evitar explicaciones enciclopédicas; una lección debe producir una sola victoria observable.
